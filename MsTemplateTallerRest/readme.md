# MSNestJSTemplateIntegrations

MSNestJSTemplateIntegrations es una plantilla para servicios NestJS en donde se compila las mejores practicas de consumo de los legados más recurrentes, adicional a esto, se define una estructura base tanto del proyecto como de sus pruebas unitarias.

## Instalación

Utilice el administrados de paquetes de node npm [npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) para intalar las dependencias del servicio MSNestJSTemplateIntegrations.

```bash
npm i
```

## Ejecución

Para incializar el programa se debe ejecutar en la consola bash el siguiente comando del framework NestJS
```bash
nest start 
```
Para la ejecución en modo desarrollo se realiza el siguiente comando
```bash
nest start --watch
```

## Contribuciones

Esta plantilla es una herramienta que nos permitirá estandarizar el código repetitivo, fue implementada teniendo en cuenta los diferentes escenarios experimentados con el equipo de trabajo, por lo cual algunos verán código de ellos reflejado en esta plantilla, por ende cualquier contribución será bienvenida, se recomienda que previo al push de los cambios se socialicen estos con el equipo para dar el visto bueno en conjunto
